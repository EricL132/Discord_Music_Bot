# Discord Music Bot

config.json  
```
{
  "Token":"TokenOfYourDiscordBot"
}  
```
Install all required dependencies  
> npm install  

Run Program  
> node index.js  

# Discord Commands
Play Song  
>-p https://www.youtube.com/watch?v=video

Pause Song
>-pause 

Resume Song
>-resume 

Skip Song
>-skip
